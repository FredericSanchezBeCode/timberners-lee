##Pourquoi
Le travail a été de reproduire une page sur Timberners-Lee basée sur un exemple donné par le coach. Celà avait comme objectif d'utilisé la majorité des compétences apprises précédemment principalement les propriétés flex et Grid.
##Comment
Pour reproduire la page, on a du créer un dossier image, une page index.html ainsi que 2pages css respectivement style.css et reset.css. Une calligraphie de font google a aussi été utilisé sur base d'une url fourni par l'énoncé. Les icones pour Facebook Twitter et Linkedin eux ont été récupérer vià la key font-awesome.
###Qui
Frédéric Sanchez Falcon étudiant à BeCode en parcours HTML/CSS
##Les difficultés et solutions.
Ici les principales difficultés ont été des problèmes d'alignement avec les grid et flex. La résolution parfois simple parfois difficile s'est faite vià des recherches sur le web ainsi que différents essais d'application de align-self;justify-content etc.
